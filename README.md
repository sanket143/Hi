# :wave: Hi-Script

Programming Language primarily for Web Scraping.

## Installation
```sh
$ make
$ sudo make install
```

## Hello World
```sh
$ hi examples/hello.hi
Hello World
```

see more [examples](https://github.com/sanket143/Hi/tree/master/examples)
